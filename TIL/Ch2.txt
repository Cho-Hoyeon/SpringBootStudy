Spring : 엔터프라이즈 App 개발 보조 프레임워크

Spring 특징 4가지 
IoC : 제어의 역전
    a = new A(); 가 아닌 private A a; 로 외부에서 객체를 받아와서 사용

DI : 의존성 주입, IoC 구현을 위해 사용
    @Autowired를 통해 Spring 컨테이너에서 Bean을 주입받아서 사용.
    Bean : Spring 컨테이너가 생성하고 관리하는 객체

AOP : 관점 지향 프로그래밍
    핵심 관점과 부가 관점을 기반으로 로직을 모듈화해, 핵심 관점 코드와 부가 관점 코드를 분리

PSA : 이식 가능한 서비스 추상화
    어떠한 기술을 사용하더라도, 일관적인 방식으로 처리할 수 있도록 인터페이스 지원
